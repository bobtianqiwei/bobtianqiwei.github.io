Desktop Buddha

Desktop Appilication, 2022

This is a desktop program inspired by Desktop Goose.
A translucent Buddha statue appears on the desktop when the program is running, emitting light while playing the Great Compassion Mantra, which can keep us calm.
We can also put our funny emoticons or any other pictures we want into the program to make it appear on the desktop.
